● Phase 3 Final Defense

Automated EdTech
Grading Assistant

A Synergistic Neuro-Symbolic Approach to Semantic Answer Evaluation

The Pipeline
An end-to-end Automated Short Answer Grading (ASAG) pipeline integrating OCR for handwritten student answers.
The Core Innovation
Combines Classical ML precision with Deep Learning semantics, achieving state-of-the-art grading accuracy through uncertainty-aware gating.
● The Problem

The Problem Statement & Evolution

Manual grading is subjective, inconsistent, and unscalable.

Phase 1 Failure
Lexical Brittleness
Fails when students use synonyms instead of exact keywords.
Phase 2 Failure
Topic Hallucination
Scores factually wrong answers highly if they contain correct vocabulary.
Phase 3 Goal
Build a Synergistic System
Create an architecture that solves both limitations simultaneously without compounding their weaknesses.

● Analysis

The "Topic vs. Factual" Failure Mode

SBERT measures topic proximity, not factual entailment.

Reference Answer (Correct)
"Mitochondria produce energy."
Student Answer (Incorrect)
"Mitochondria control gene expression."
Result
High Similarity (>0.55)
SBERT groups them tightly in embedding space because they share the same biological domain vocabulary, despite being biologically false.
● Phase 3

The Synergistic Hybrid Architecture

The Solution
Combine SVM precision with SBERT robustness.
Uses an information-theoretic gating mechanism rather than a naive average.
The Mechanism
if max(SVM_prob) ≥ 0.85
FAST PATH (Yes)
Return SVM instantly
If SVM is confident, skip SBERT entirely to save compute and avoid topic hallucinations.
FULL PATH (No)
Trigger SBERT Ensemble
If SVM is uncertain, trigger the semantic SBERT ensemble to resolve the decision boundary.

● System Integration

Fixing the HTR Bottleneck

The Issue
TrOCR hallucinations on paragraphs
TrOCR was trained on single cropped handwriting lines (IAM dataset). Feeding a full paragraph overwhelms the ViT cross-attention mechanism.
The Solution
Classical Horizontal Projection
Calculates row-wise pixel density to detect whitespace valleys. Crops paragraph into lines prior to neural inference, iterating sequentially.

● Analysis

The Domain Generalization Gap

SVM Vulnerability
Collapses on new vocabulary
On Unseen Domains (test_ud split), SVM performance drops to F1 = 0.31. TF-IDF memorizes domain-specific terms.
SBERT Strength
Generalizes better
Pre-trained dense embeddings allow SBERT to maintain performance across entirely new science topics.
Hybrid Impact
The hybrid model relies heavily on the SVM (α=0.85). It inherits both the SVM's precision on known domains, and its vulnerability to unseen domains.

● Ethics

Bias & Fairness Analysis

The Threat
Length Bias
AI graders often suffer from "length bias" — assigning higher grades simply because an answer is longer.
The Evaluation
Pearson Correlation: r < 0.15
Negligible correlation between word count and confidence. The model successfully grades on substance, not verbosity.
