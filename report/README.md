# Project Report Documentation

This directory contains the source files for the academic documentation of the Automated EdTech Grading Assistant.

## Contents
- **`main.tex`**: The primary LaTeX source file (IEEE conference format).
- **`MLProject_Report.pdf`**: The compiled final report documenting methodology, results, and theoretical rigorous proof.

## Academic Rigor
The report covers:
- Literature Review of ASAG (Automated Short Answer Grading) systems.
- Mathematical derivation of TF-IDF and Semantic Similarity metrics.
- Empirical results from the SciEntsBank evaluation splits (UA, UQ, UD).
- Limitation analysis and future research directions.
