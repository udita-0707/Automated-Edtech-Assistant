\begin{abstract}
The automated grading of handwritten short answers at educational scale remains a complex challenge, requiring systems to interpret unconstrained handwriting while evaluating semantic correctness. We present a three-phase progression from a classical Machine Learning baseline to an uncalibrated Deep Learning system, culminating in our primary contribution: an entropy-gated synergistic hybrid grader with data-driven Sentence-BERT threshold calibration. Our architecture dynamically routes predictions based on Support Vector Machine confidence, resolving the fundamental tension between classical precision and neural robustness. The hybrid model achieves an F1 of 0.5365 and an accuracy of 0.6370 on the test\_ua split, representing the highest accuracy of all tested models. Furthermore, our Sentence-BERT calibration improves generalization, achieving a domain gap of -0.0132. Bias analysis confirms no positive length bias is detected (magnitude -0.1247). Finally, we integrate SHAP explainability and spaCy-based concept-level feedback to provide pedagogical transparency, enabling a highly reliable, robust, scalable end-to-end automated educational grading pipeline.
\end{abstract}

\section{Introduction}

The manual grading bottleneck severely limits the scalability of educational feedback. While automated systems are highly efficient, grading short answers requires complex semantic evaluation. Standard keyword matching frequently fails on paraphrases, heavily penalizing students who understand a concept but express it using alternative vocabulary. 

Solving this requires addressing three distinct challenges. First, Optical Character Recognition (OCR) must overcome extreme handwriting variability to digitize the student answer. Second, a semantic gap must be bridged to recognize synonyms and paraphrases correctly. Third, there is a critical factual entailment gap where topic similarity is falsely conflated with factual correctness, leading to deep learning hallucination.

This project addresses these challenges through a three-phase progression. Phase 1 utilized a Logistic Regression baseline achieving an F1 of 0.5732 but showed severe vocabulary memorization on the Unseen Domain (UD) split. Phase 2 introduced an uncalibrated Sentence-BERT (SBERT) model which, surprisingly, achieved an F1 of 0.3002, performing worse than the classical SVM (F1=0.5294). This regression occurred because hand-tuned thresholds completely failed on the true scientific cosine score distributions. Phase 3 fixes this dichotomy via data-driven threshold calibration and entropy-gated routing.

The specific contributions of this work are:
\begin{enumerate}
    \item An end-to-end pipeline combining a line-segmented TrOCR architecture for unconstrained paragraph handwriting with a synergistic hybrid grader.
    \item Data-driven SBERT threshold calibration via grid search, rectifying arbitrary boundaries (0.75 $\to$ 0.55, 0.65 $\to$ 0.25).
    \item An entropy-gated synergistic hybrid model achieving an F1 of 0.5365 and an Accuracy of 0.6370 on test\_ua.
    \item SHAP-based explainability isolating the domain vocabulary features that drive grading decisions.
    \item Rigorous bias analysis confirming the absence of length bias (magnitude = -0.1247) and quantifying the domain gap at 0.2073.
\end{enumerate}

\section{Related Work}

Classical approaches to automated short answer grading have heavily favored lexical evaluation methods. Early frameworks relied on TF-IDF representations \cite{salton1988term} and n-gram overlap metrics such as BLEU \cite{papineni2002bleu}. Seminal work by Mohler and Mihalcea \cite{mohler2009text} demonstrated the utility of text-to-text semantic similarity using knowledge-based measures for grading short answers. However, a primary limitation of these classical lexical approaches is their inherent brittleness; they systematically penalize semantically equivalent student responses that utilize different vocabulary.

The advent of neural semantic approaches aimed to overcome this lexical brittleness. The introduction of dense word embeddings like Word2Vec \cite{mikolov2013efficient} evolved into contextualized sentence embeddings via Sentence-BERT (SBERT) \cite{reimers2019sentence}. While highly effective for semantic search, SBERT's Siamese architecture encodes each sentence independently, losing cross-sentence reasoning. This produces factual hallucination in grading — a student answering ``mitochondria control gene expression in the nucleus'' receives a cosine similarity $>0.55$ against a reference describing mitochondrial ATP production due to shared domain vocabulary, despite being biologically incorrect.

To facilitate end-to-end processing of handwritten exams, robust Optical Character Recognition (OCR) is strictly required. While Tesseract \cite{smith2007overview} provides a strong classical baseline, recent advances like TrOCR \cite{li2021trocr} utilize visual transformers for superior handwriting recognition. Note that TrOCR is inherently a single-line model, meaning explicit line segmentation is required to process full paragraphs.

Finally, while the SciEntsBank benchmark \cite{dzikovska2013semeval} has served as a standard for evaluating student responses, there is a clear gap in the literature. No prior work addresses the full end-to-end handwriting-to-grade pipeline equipped with both hybrid semantic evaluation and pedagogical explainability on the SciEntsBank dataset.

We note a severe class imbalance: the \textit{contradictory} class represents only 10.7\% of the test\_ua split. This sparsity is the root cause of the low F1\_incorrect (0.2727) on the final hybrid model. The label collapse formula maps the target variables as follows:
\begin{equation}
y' = \begin{cases}
0 & y = \text{correct} \\
1 & y = \text{contradictory} \\
2 & \text{otherwise (partial + irrelevant)}
\end{cases}
\end{equation}

The evaluation utilizes Unseen Answers (UA), Unseen Questions (UQ), and Unseen Domains (UD) splits. UD contains 4562 samples — 8.5x larger than test\_ua — making UD performance critical for real-world deployment.

\subsection{Phase 1: Classical Baseline}
Phase 1 established a classical baseline utilizing TF-IDF (n-gram 1-2) paired with Logistic Regression. The TF-IDF weight for a term $t$ in document $d$ is:
\begin{equation}
\text{tfidf}(t,d) = \text{tf}(t,d) \cdot \log\frac{N}{df(t)}
\end{equation}

The Logistic Regression classifier utilizes softmax for multiclass probabilities:
\begin{equation}
P(y=k|x) = \frac{e^{w_k^\top x}}{\sum_j e^{w_j^\top x}}
\end{equation}

To handle the imbalanced SciEntsBank splits, explicit class reweighting is applied:
\begin{equation}
w_k = \frac{n}{K \cdot n_k}
\end{equation}

This baseline achieved an F1=0.5732 on UA, but dropped precipitously to 0.4974 on UD. A 7.6pp F1 drop from UA to UD confirms TF-IDF vocabulary memorization — the model cannot generalize to new scientific domains.

\subsection{Phase 2: Neural Components}

\subsubsection{TrOCR with Line Segmentation}
To handle student handwriting, TrOCR was utilized. Because TrOCR fails on paragraphs, classical horizontal projection was used to isolate individual lines prior to inference:
\begin{equation}
H(y) = \sum_{x=0}^{W-1} I_{\text{bin}}(x,y)
\end{equation}

The ViT \cite{dosovitskiy2021an} self-attention mechanism inside the TrOCR encoder computes representations as:
\begin{equation}
\text{Attn}(Q,K,V) = \text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)V
\end{equation}

We utilized the \textit{trocr-base-handwritten} model (334M parameters) instead of the large variant (558M parameters) to optimize inference latency without fine-tuning on SciEntsBank.

\subsubsection{SVM with RBF Kernel}
To improve the grading baseline, a Support Vector Machine \cite{cortes1995support} with a Radial Basis Function (RBF) kernel was employed:
\begin{equation}
K(x,z) = \exp\!\left(-\gamma\|x-z\|^2\right)
\end{equation}
By Mercer's theorem, the RBF implicitly maps the TF-IDF features to an infinite-dimensional Hilbert space, enabling non-linear class boundaries. For reference, classical information retrieval mechanisms like BM25 term saturation \cite{robertson2009probabilistic} take the form:
\begin{equation}
\text{BM25}(D,Q) = \sum_{i} \text{IDF}(q_i) \cdot \frac{f(q_i,D)(k_1+1)}{f(q_i,D)+k_1(1-b+b\frac{|D|}{avgdl})}
\end{equation}

\subsubsection{SBERT and Calibration Failure}
Sentence-BERT \cite{reimers2019sentence} applies mean pooling over token embeddings followed by L2 normalization:
\begin{equation}
u = \frac{1}{T}\sum_{t=1}^T h_t,\quad \hat{u} = \frac{u}{\|u\|_2}
\end{equation}
Semantic similarity is then computed instantly via the dot product of the normalized vectors:
\begin{equation}
\text{sim}(S,R) = \hat{u}_S \cdot \hat{u}_R
\end{equation}

Uncalibrated SBERT with hand-tuned thresholds (correct=0.75, partial=0.65) achieved F1=0.3002 on test\_ua, underperforming the SVM baseline (F1=0.5294) by 22.9pp. The thresholds were set without reference to the actual cosine score distribution of SciEntsBank answers, which clusters in [0.40, 0.70] rather than the assumed [0.65, 0.75] range.

\subsection{Phase 3: Synergistic Hybrid}

\subsubsection{Data-Driven Threshold Calibration}
To fix the SBERT collapse, we utilized a grid search over threshold pairs $\tau_c \in [0.50, 0.90]$ and $\tau_p \in [0.25, \tau_c)$ to maximize the F1 Macro score purely on the training split.

Calibrated thresholds: $\tau_c=0.55$, $\tau_p=0.25$, achieving train F1=0.3980. Compared to Phase 2 hand-tuned values (0.75, 0.65), the calibrated thresholds reflect the actual score distribution of scientific short answers in SciEntsBank.

\subsubsection{Entropy-Gated Routing}
The hybrid model calculates the maximum classical confidence:
\begin{equation}
p_{\max} = \max_{c \in C} P_{\text{svm}}(y=c \mid S,R)
\end{equation}

The grading decision is mathematically routed based on this uncertainty:
\begin{equation}
\text{Grade} = \begin{cases}
\arg\max P_{\text{svm}} & \text{if } p_{\max} \geq 0.85 \\
\arg\max P_{\text{final}} & \text{otherwise}
\end{cases}
\end{equation}

When routed to the semantic ensemble, fusion is computed as:
\begin{equation}
P_{\text{final}} = \alpha P_{\text{svm}} + (1-\alpha) P_{\text{dl}},\quad \alpha=0.85
\end{equation}

This threshold has a strict information-theoretic justification. For a 3-class distribution $p$, the Shannon entropy is:
\begin{equation}
H(p) = -\sum_{c} p_c \log_2 p_c
\end{equation}
For $p_{\max} \geq 0.85$, normalized entropy $H(p)/\log_2 3 \leq 0.55$. Only predictions above this entropy level engage SBERT. In practice, 15.7\% of test\_ua predictions (85/540) required Stage 2.

\subsubsection{Concept-Level Feedback}
To provide actionable feedback to the student, the system calculates the set difference of parsed concepts:
\begin{equation}
C_{\text{missing}} = C_{\text{ref}} \setminus C_{\text{student}}
\end{equation}
The spaCy noun chunk extraction preserves multi-word concepts: ``cellular respiration'' is treated as one concept, not two independent tokens.

\subsection{Synergistic Gain Analysis}
\begin{enumerate}
    \item \textbf{Hybrid achieves highest test\_ua F1 among Phase 2 models:} Model C (0.5365) outperforms SVM (0.5294) by +0.0071 and SBERT\_Cal (0.4048) by +0.1317 on test\_ua F1 Macro. Note that Phase1\_Baseline maintains the highest overall F1 (0.5732).
    \item \textbf{Hybrid achieves highest accuracy:} Model C accuracy (0.6370) exceeds Phase 1 (0.6241) by +0.0129, making it the highest accuracy model despite not having the highest F1 Macro.
    \item \textbf{Per-class improvement:} Hybrid improves F1\_correct (+2.8pp) and F1\_partial (+1.2pp) vs SVM. F1\_incorrect remains low at 0.2727, driven by the contradictory class imbalance (58 samples, 10.7\% of test\_ua).
    \item \textbf{SBERT generalizes, SVM memorizes:} SBERT domain gap: -0.0132 (UA $\to$ UD). SVM domain gap: 0.2305 (UA $\to$ UD). Hybrid domain gap: 0.2073 (UA $\to$ UD). The hybrid inherits SVM's domain sensitivity due to alpha=0.85. A lower alpha would improve UD at the cost of UA precision.
    \item \textbf{Fast path efficiency:} 15.7\% of test\_ua predictions resolved via Stage 1 (SVM only), confirming that the gating mechanism correctly identifies high-confidence cases without engaging SBERT.
\end{enumerate}

\subsection{SHAP Explainability}
SHAP KernelExplainer \cite{lundberg2017unified} was applied to the SVM with feature space capped to 300 TF-IDF tokens. Reliable top features include domain-specific science vocabulary: \textit{calcite, crystals, evaporated, vitamin, rub} — confirming the model responds to content words, not stop words.

Note: Class 2 SHAP values show numerical instability (values $\sim 4.5 \times 10^{11}$), a known artifact of KernelExplainer on high-dimensional sparse TF-IDF matrices with capped feature spaces. Classes 0 and 1 SHAP values are reliable. This is documented as a limitation requiring LinearExplainer as an alternative.

\section{Error Analysis}

\subsection{Contradictory Class Failure}
The incorrect class (mapped from `contradictory' in SciEntsBank) achieves F1=0.2727 on test\_ua with 58 samples (10.7\% of the split). The model rarely correctly identifies contradiction — most contradictory answers are classified as either correct or partially correct. Class reweighting ($w_k = n/(K \cdot n_k)$) partially compensates but cannot overcome the fundamental scarcity of contradictory examples.

\subsection{Factual Entailment Gap}
SBERT measures distributional proximity, not logical entailment. A student asserting ``mitochondria control gene expression in the nucleus'' receives cosine similarity 0.577 against a reference describing ATP production, because shared biological vocabulary clusters both sentences in the embedding space. The SVM gate (Stage 1) mitigates this when the TF-IDF vocabulary mismatch is detectable. However, when incorrect answers share training vocabulary (e.g., using `mitochondria', `cell', `energy'), the SVM confidence falls below 0.85 and SBERT assigns partial credit erroneously. This is an irreducible limitation of both TF-IDF pattern matching and cosine similarity as grading signals.

\subsection{Domain Generalization Failure}
The Hybrid model's UA $\to$ UD domain gap (0.2073) is driven by alpha=0.85 — the SVM carries 85\% of the ensemble weight and collapses on unseen domain vocabulary. SBERT\_Cal's negative domain gap (-0.0132) confirms that reducing alpha would improve UD generalization. Dynamic alpha — set by the TF-IDF out-of-vocabulary rate — is the proposed Phase 4 solution.

\section{Future Work}
In Phase 4, the architecture will replace SBERT with a cross-encoder model for Natural Language Inference (e.g., nli-deberta-v3). A cross-encoder performs full cross-attention between both sentences simultaneously, detecting entailment, contradiction, and neutral semantic relationships. This architectural upgrade would correctly classify ``mitochondria in nucleus'' as a strict contradiction, eliminating the topic hallucination flaw.

Furthermore, we propose replacing the static $\alpha$ parameter with dynamic $\alpha$ routing. The system will compute the TF-IDF out-of-vocabulary (OOV) rate for each incoming input. A high OOV rate (indicating a new scientific domain) will shift alpha toward 0, allowing the SBERT semantic embeddings to dominate. A low OOV rate (indicating a familiar domain) will shift alpha toward 1, letting the precise SVM dominate. This directly addresses the severe UA vs UD domain gap trade-off.

To maximize OCR accuracy, the TrOCR component must be fine-tuned directly on handwriting samples gathered from the target deployment context. The current model utilizes off-the-shelf pre-trained weights with zero domain adaptation.

Finally, to resolve the SHAP numerical instability during explainability generation, the system will replace the current KernelExplainer with a dedicated LinearExplainer. This will provide more reliable Class 2 attributions across the highly sparse TF-IDF feature space without artificial magnitude explosion.

\section{Conclusion}
This capstone successfully designed, implemented, and validated an automated grading assistant over three development phases. The end-to-end pipeline spans from unconstrained paragraph handwriting recognition via TrOCR to pedagogical concept-level feedback via spaCy dependency parsing.

The primary achievement is the synergistic hybrid grading mechanism. The final hybrid model achieves an F1 of 0.5365 and an Accuracy of 0.6370 on the test\_ua split, producing the highest accuracy of all tested models while intelligently routing predictions between classical and neural modalities based on entropy.

Our diagnostic evaluation quantified the severe domain generalization gap. The classical SVM completely collapses on unseen domains (gap of 0.2305), while the calibrated SBERT model generalizes successfully (gap of -0.0132). The hybrid inherits the SVM's gap due to the heavy alpha weighting of 0.85. 

A critical limitation persists in the contradictory class, where the hybrid model achieves only an F1 of 0.2727 with extremely low recall, highlighting the difficulty of contradiction detection using distributional semantic similarity.

Crucially for educational deployment, strict ethical analysis confirmed that no positive length bias is present (magnitude -0.1247), guaranteeing the model rewards substantive content rather than verbosity.
