# Automated EdTech Frontend

This is a modern React application built with **Vite** and **Tailwind CSS v4**, serving as the user interface for the Automated EdTech Grading Assistant.

## Features
- **Real-time Grading**: Submit student answers against reference answers for instant feedback.
- **Microservice Integration**: Connects to the Node.js backend to orchestrate grading requests.
- **Dynamic State**: Uses React hooks for efficient state management and UI updates.
- **Responsive UI**: Styled with Tailwind CSS for a seamless experience across devices.

## Setup
1. Standard install:
   ```bash
   npm install
   ```
2. Development server:
   ```bash
   npm run dev
   ```

## Design System
The frontend follows a clean, "EdTech-professional" aesthetic with a focus on readability and accessibility.
