Context

The Grading Challenge

Automated Grading?

Evaluating student responses using software to provide instant, consistent feedback at any scale.

Why it's Important

Scalability in MOOCs and digital classrooms. Fairness by eliminating human fatigue and subjective bias.

Key Challenges

Semantic variability (different words, same meaning) and logical understanding of scientific concepts.

Data Engineering

SciEntsBank Corpus

A benchmark dataset from SemEval-2013 Task 7 for student response evaluation.

  • 135 Unique Science Questions
  • ~10,000 High-Quality Student Responses
  • 3-Way Labeling: Correct, Incorrect, Partial
  • Expert-verified reference answers

Data Splits (UA, UQ, UD)

UA: Unseen Answers

UQ: Unseen Questions

UD: Unseen Domains

Insights

Data Visual Analysis

Decoding patterns in human grading behavior.

Labels

Imbalance detection: Incorrect class dominance requires balanced loss weights.

Semantic Proof

Cluster analysis: Mathematical evidence that answers separate by correctness.

Length Pattern

Correlation: Significant overlap between answer length and grade quality.

Feature Engineering

Integrated Feature Space

TF-IDF Vectors

Capturing the importance of specific terms within the scientific context.

Semantic Indices

Jaccard Similarity and Token Density ratios against reference standards.

Structural Metrics

Answer/Reference length ratio and word density metrics.

Machine Learning CORE

Logistic Regression

A supervised linear classifier chosen for its extreme Explainability.

z = w1xtfidf + w2xsemantic + w3xstructural + b
  • Multi-class one-vs-rest strategy
  • L2 Regularization to prevent overfitting
  • Balanced weights for handling skewness

Advantages

Faster than BERT models (~1ms inference time). Allows teachers to see "why" a student got a specific grade based on feature weights.

Conclusion

Project Summary

Limitations

  • Vocabulary sensitivity (Exact match dependent)
  • No OCR (Digital text only)
  • Lacks deep semantic negation logic

Future Work

  • Sentence-Transformer Embeddings
  • Hybrid Deep Learning Models (BERT)
  • OCR Handwriting Integration

Final Conclusion

We successfully built a robust baseline for automated EdTech grading using only mathematical manifolds and classical regression, proving that explainability and scalable feedback can coexist.
