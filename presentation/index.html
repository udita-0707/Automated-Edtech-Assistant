Problem Statement

Solving the Grading Bottleneck

Current educational systems struggle with manual evaluation of descriptive answers, leading to fatigue and inconsistency.

The Goal

Build a transparent, math-grounded system that understands student semantics without the high cost of Large Language Models.

Instant

Provides sub-second feedback for entire classrooms.

Fair

Eliminates subjective human bias via mathematical consistency.

EDA Insights

Visual Evidence & Patterns

Analyzing the SciEntsBank dataset (10,000+ samples) to drive feature engineering.

1. Label Distribution

Skewed training data required "Balanced Class Weights" to ensure the model doesn't ignore minority labels.

2. Semantic Manifold

PCA visualization proof: Students' correct and incorrect answers form distinct mathematical clusters.

3. Feature Interaction

Correlation map confirms TF-IDF features and Structural features are non-redundant.

4. Length Impact

Discovery: Answer length is a significant secondary signal for "Correctness" in scientific responses.
